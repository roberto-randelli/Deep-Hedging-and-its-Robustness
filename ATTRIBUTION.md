# Attribution

## Replication Code

The repository contains code inspired from the following repository, used for replication purposes:

- **Repository**: https://github.com/Guangyi-Mira/Distributional-Adversarial-Attacks-and-Training-in-Deep-Hedging
- **Authors**: Guangyi He, Tobias Sutter, Lukas Gonon
- **Paper**: He, G., Sutter, T., & Gonon, L. (2025). Distributional Adversarial Attacks
  and Training in Deep Hedging. arXiv:2508.14757v2. NeurIPS 2025.
- **License**: MIT License (see `Distributional-Adversarial-Attacks-and-Training-in-Deep-Hedging/LICENSE`)

This code is included solely for academic replication and comparison purposes
as part of an MSc dissertation. All credit for this implementation belongs to the
original authors.

## Novel Contributions (Phase 2 — Extension)

All code in `src/novelty` is original work by Roberto Randelli, written from scratch for the MSc
Mathematical Finance dissertation at the University of Oxford, 2026.

The extension code in `src/` does not import, copy, or adapt any file from the cloned
repository above. It reimplements the deep hedging framework independently, using
He et al. (2025) and Buehler et al. (2019) as mathematical references only.
